几何公式

欧氏几何

平面图形面积

\[ \text{三角形: } A = \frac{1}{2}bh \]
\[ \text{矩形: } A = lw \]
\[ \text{正方形: } A = s^2 \]
\[ \text{平行四边形: } A = bh \]
\[ \text{梯形: } A = \frac{1}{2}(a+b)h \]
\[ \text{圆: } A = \pi r^2 \]
\[ \text{椭圆: } A = \pi ab \]
\[ \text{扇形: } A = \frac{1}{2}r^2\theta \]

常见平面图形的面积公式，其中 \(b\) 为底，\(h\) 为高，\(l\) 为长，\(w\) 为宽，\(s\) 为边长，\(r\) 为半径，\(a,b\) 为半轴长，\(\theta\) 为圆心角（弧度）。

立体图形体积

\[ \text{长方体: } V = lwh \]
\[ \text{正方体: } V = s^3 \]
\[ \text{圆柱体: } V = \pi r^2h \]
\[ \text{圆锥体: } V = \frac{1}{3}\pi r^2h \]
\[ \text{球体: } V = \frac{4}{3}\pi r^3 \]
\[ \text{棱锥体: } V = \frac{1}{3}Bh \]
\[ \text{椭球体: } V = \frac{4}{3}\pi abc \]
\[ \text{圆台: } V = \frac{1}{3}\pi h(r^2 + Rr + R^2) \]

常见立体图形的体积公式，其中 \(l\) 为长，\(w\) 为宽，\(h\) 为高，\(s\) 为边长，\(r\) 为底面半径，\(R\) 为顶面半径，\(B\) 为底面积，\(a,b,c\) 为半轴长。

勾股定理

\[ a^2 + b^2 = c^2 \]

勾股定理（毕达哥拉斯定理）：在直角三角形中，两直角边（\(a\) 和 \(b\)）的平方和等于斜边（\(c\)）的平方。

三角形定理

\[ \text{正弦定理: } \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R \]
\[ \text{余弦定理: } c^2 = a^2 + b^2 - 2ab\cos C \]
\[ \text{面积公式: } A = \frac{1}{2}ab\sin C = \frac{1}{2}bc\sin A = \frac{1}{2}ac\sin B \]
\[ \text{海伦公式: } A = \sqrt{s(s-a)(s-b)(s-c)}, \text{ 其中 } s = \frac{a+b+c}{2} \]

三角形的重要定理，其中 \(a,b,c\) 为三角形的边长，\(A,B,C\) 分别为对应边的对角，\(R\) 为外接圆半径，\(s\) 为半周长。

解析几何

坐标系

\[ \text{笛卡尔坐标: } (x, y, z) \]
\[ \text{极坐标: } (r, \theta), \text{ 其中 } x = r\cos\theta, y = r\sin\theta \]
\[ \text{柱坐标: } (r, \theta, z), \text{ 其中 } x = r\cos\theta, y = r\sin\theta \]
\[ \text{球坐标: } (r, \theta, \phi), \text{ 其中 } x = r\sin\phi\cos\theta, y = r\sin\phi\sin\theta, z = r\cos\phi \]

常见坐标系及其转换关系，其中 \(r\) 为径向距离，\(\theta\) 为方位角，\(\phi\) 为极角。

直线和平面

\[ \text{直线点斜式: } y - y_0 = m(x - x_0) \]
\[ \text{直线截距式: } \frac{x}{a} + \frac{y}{b} = 1 \]
\[ \text{直线一般式: } Ax + By + C = 0 \]
\[ \text{平面一般式: } Ax + By + Cz + D = 0 \]
\[ \text{平面点法式: } A(x - x_0) + B(y - y_0) + C(z - z_0) = 0 \]

直线和平面的各种表示形式，其中 \(m\) 为斜率，\((x_0, y_0, z_0)\) 为点坐标，\(a,b\) 为截距，\(A,B,C,D\) 为常数。

二次曲线

\[ \text{圆: } (x - h)^2 + (y - k)^2 = r^2 \]
\[ \text{椭圆: } \frac{(x - h)^2}{a^2} + \frac{(y - k)^2}{b^2} = 1 \]
\[ \text{双曲线: } \frac{(x - h)^2}{a^2} - \frac{(y - k)^2}{b^2} = 1 \text{ 或 } \frac{(y - k)^2}{a^2} - \frac{(x - h)^2}{b^2} = 1 \]
\[ \text{抛物线: } y = a(x - h)^2 + k \text{ 或 } x = a(y - k)^2 + h \]

二次曲线的标准方程，其中 \((h, k)\) 为中心或顶点坐标，\(a,b\) 为半轴长，\(r\) 为半径。

距离公式

\[ \text{两点间距离: } d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2 + (z_2 - z_1)^2} \]
\[ \text{点到直线距离: } d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}} \]
\[ \text{点到平面距离: } d = \frac{|Ax_0 + By_0 + Cz_0 + D|}{\sqrt{A^2 + B^2 + C^2}} \]
\[ \text{两平行线距离: } d = \frac{|C_2 - C_1|}{\sqrt{A^2 + B^2}} \]

各种距离公式，其中 \((x_0, y_0, z_0)\) 为点坐标，\(Ax + By + Cz + D = 0\) 为平面方程，\(Ax + By + C = 0\) 为直线方程。

微分几何

曲线几何

\[ \text{弧长: } s = \int_{a}^{b} \sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2 + \left(\frac{dz}{dt}\right)^2} dt \]
\[ \text{曲率: } \kappa = \frac{|\mathbf{T}'(t)|}{|\mathbf{r}'(t)|} = \frac{|\mathbf{r}'(t) \times \mathbf{r}''(t)|}{|\mathbf{r}'(t)|^3} \]
\[ \text{挠率: } \tau = \frac{(\mathbf{r}'(t) \times \mathbf{r}''(t)) \cdot \mathbf{r}'''(t)}{|\mathbf{r}'(t) \times \mathbf{r}''(t)|^2} \]
\[ \text{法向量: } \mathbf{N}(t) = \frac{\mathbf{T}'(t)}{|\mathbf{T}'(t)|} \]
\[ \text{副法向量: } \mathbf{B}(t) = \mathbf{T}(t) \times \mathbf{N}(t) \]

空间曲线的微分几何量，其中 \(\mathbf{r}(t)\) 为曲线的参数方程，\(\mathbf{T}(t)\) 为单位切向量，\(\kappa\) 为曲率，\(\tau\) 为挠率。

曲面几何

\[ \text{第一基本形式: } I = E du^2 + 2F du dv + G dv^2 \]
\[ \text{第二基本形式: } II = L du^2 + 2M du dv + N dv^2 \]
\[ \text{高斯曲率: } K = \frac{LN - M^2}{EG - F^2} \]
\[ \text{平均曲率: } H = \frac{EN - 2FM + GL}{2(EG - F^2)} \]
\[ \text{法曲率: } \kappa_n = \frac{II}{I} = \frac{L du^2 + 2M du dv + N dv^2}{E du^2 + 2F du dv + G dv^2} \]

曲面的微分几何量，其中 \(E, F, G\) 为第一基本形式的系数，\(L, M, N\) 为第二基本形式的系数，\(K\) 为高斯曲率，\(H\) 为平均曲率。
