数论公式

基础数论

整除性

\[ a \mid b \iff \exists k \in \mathbb{Z}, b = ak \]
\[ a \mid b \text{ 且 } a \mid c \implies a \mid (b + c) \]
\[ a \mid b \implies a \mid bc \text{ 对任意 } c \in \mathbb{Z} \]
\[ a \mid b \text{ 且 } b \mid c \implies a \mid c \]

整除的基本性质。符号 \(a \mid b\) 表示a整除b，即b是a的倍数。

最大公约数和最小公倍数

\[ \gcd(a, b) = \min \{ d \in \mathbb{N} : d \mid a \text{ 且 } d \mid b \} \]
\[ \text{lcm}(a, b) = \max \{ m \in \mathbb{N} : a \mid m \text{ 且 } b \mid m \} \]
\[ a \cdot b = \gcd(a, b) \cdot \text{lcm}(a, b) \]
\[ \gcd(a, b) = \gcd(b, a \mod b) \quad (\text{欧几里得算法}) \]

最大公约数(gcd)和最小公倍数(lcm)的定义及性质。欧几里得算法是计算最大公约数的有效方法。

素因数分解

\[ n = p_1^{e_1} p_2^{e_2} \cdots p_k^{e_k} \]
\[ d(n) = (e_1 + 1)(e_2 + 1) \cdots (e_k + 1) \quad (\text{因数个数}) \]
\[ \sigma(n) = \frac{p_1^{e_1+1} - 1}{p_1 - 1} \cdot \frac{p_2^{e_2+1} - 1}{p_2 - 1} \cdots \frac{p_k^{e_k+1} - 1}{p_k - 1} \quad (\text{因数和}) \]

正整数的素因数分解及相关函数。其中 \(p_1, p_2, \ldots, p_k\) 是不同的素数，\(e_1, e_2, \ldots, e_k\) 是正整数。

贝祖定理

\[ \forall a, b \in \mathbb{Z}, \exists x, y \in \mathbb{Z}, ax + by = \gcd(a, b) \]

贝祖定理指出，任意两个整数的最大公约数可以表示为这两个整数的线性组合。

素数理论

素数定义

\[ p \text{ 是素数} \iff \text{只有 } 1 \text{ 和 } p \text{ 能整除 } p \]

素数的定义：大于1的自然数，除了1和它本身外，不能被其他自然数整除的数。

素数定理

\[ \pi(x) \sim \frac{x}{\ln x} \]
\[ \lim_{x \to \infty} \frac{\pi(x) \ln x}{x} = 1 \]

素数定理描述了小于等于x的素数个数π(x)的渐近行为。它表明π(x)大约等于x/ln x。

埃拉托斯特尼筛法

1. 列出从2到n的所有整数

2. 从列表中找到最小的未标记的数p，它是素数

3. 标记p的所有倍数（2p, 3p, ...）

4. 重复步骤2和3，直到所有数都被标记或检查过

5. 未被标记的数都是素数

埃拉托斯特尼筛法是一种古老而有效的找出小于等于给定整数n的所有素数的算法。

梅森素数

\[ M_p = 2^p - 1 \]
\[ \text{若 } M_p \text{ 是素数，则 } p \text{ 必须是素数} \]

梅森素数是形如Mₚ = 2ᵖ - 1的素数，其中p本身是素数。注意，并不是所有素数p都能使Mₚ成为素数。

费马素数

\[ F_n = 2^{2^n} + 1 \]
\[ F_0 = 3, F_1 = 5, F_2 = 17, F_3 = 257, F_4 = 65537 \]

费马素数是形如Fₙ = 2²ⁿ + 1的素数。目前已知的费马素数只有F₀到F₄五个。

同余理论

同余定义

\[ a \equiv b \pmod{m} \iff m \mid (a - b) \]

同余的定义：如果a和b除以m的余数相同，则称a和b模m同余，记作a ≡ b (mod m)。

同余性质

\[ a \equiv a \pmod{m} \quad (\text{自反性}) \]
\[ a \equiv b \pmod{m} \implies b \equiv a \pmod{m} \quad (\text{对称性}) \]
\[ a \equiv b \pmod{m} \text{ 且 } b \equiv c \pmod{m} \implies a \equiv c \pmod{m} \quad (\text{传递性}) \]
\[ a \equiv b \pmod{m} \text{ 且 } c \equiv d \pmod{m} \implies a + c \equiv b + d \pmod{m} \quad (\text{加法保持}) \]
\[ a \equiv b \pmod{m} \text{ 且 } c \equiv d \pmod{m} \implies ac \equiv bd \pmod{m} \quad (\text{乘法保持}) \]
\[ a \equiv b \pmod{m} \implies ka \equiv kb \pmod{m} \quad (\text{数乘保持}) \]

同余关系的基本性质，包括自反性、对称性、传递性以及加法、乘法和数乘的保持性。

欧拉定理

\[ \text{若 } \gcd(a, m) = 1 \text{，则 } a^{\phi(m)} \equiv 1 \pmod{m} \]

欧拉定理：如果a和m互质，则a的欧拉函数φ(m)次方模m等于1。其中φ(m)是小于等于m且与m互质的正整数的个数。

费马小定理

\[ \text{若 } p \text{ 是素数且 } p \nmid a \text{，则 } a^{p-1} \equiv 1 \pmod{p} \]
\[ \text{对任意整数 } a \text{，有 } a^p \equiv a \pmod{p} \]

费马小定理：如果p是素数且a不被p整除，则a的p-1次方模p等于1。这是欧拉定理的一个特例，当m是素数p时，φ(p) = p-1。

中国剩余定理

设m₁, m₂, ..., mₖ是两两互质的正整数，则同余方程组：

\[ x \equiv a_1 \pmod{m_1} \]
\[ x \equiv a_2 \pmod{m_2} \]
\[ \vdots \]
\[ x \equiv a_k \pmod{m_k} \]

在模M = m₁m₂⋯mₖ下有唯一解：

\[ x \equiv \sum_{i=1}^{k} a_i M_i N_i \pmod{M} \]

其中Mᵢ = M/mᵢ，Nᵢ是Mᵢ模mᵢ的逆元，即MᵢNᵢ ≡ 1 (mod mᵢ)

中国剩余定理（孙子定理）：如果m₁, m₂, ..., mₖ是两两互质的正整数，那么同余方程组在模M = m₁m₂⋯mₖ下有唯一解。

丢番图方程

一次丢番图方程

方程形式

\[ ax + by = c \]

解的存在性

\[ \text{方程有整数解} \iff \gcd(a,b) \mid c \]

通解公式

\[ x = x_0 + \frac{b}{\gcd(a,b)} t, \quad y = y_0 - \frac{a}{\gcd(a,b)} t, \quad t \in \mathbb{Z} \]

一次丢番图方程ax + by = c的求解理论，包括解的存在性条件和通解公式，其中(x₀,y₀)是方程的一个特解。

佩尔方程

方程形式

\[ x^2 - Dy^2 = 1 \]

基本解与通解

\[ (x_n + y_n\sqrt{D}) = (x_1 + y_1\sqrt{D})^n, \quad n \in \mathbb{N} \]

示例：Pell(2)

\[ x^2 - 2y^2 = 1, \quad \text{基本解: } (3,2), \text{ 下一个解: } (17,12) \]

佩尔方程x² - Dy² = 1是一类重要的二次丢番图方程，当D是正非平方整数时，方程有无穷多组正整数解。

费马大定理

\[ x^n + y^n = z^n \]
\[ \text{当 } n > 2 \text{ 时，没有正整数解} \]

费马大定理是数论中最著名的定理之一，由安德鲁·怀尔斯于1994年证明。它指出，当n>2时，方程xⁿ + yⁿ = zⁿ没有正整数解。

解析数论

黎曼ζ函数

定义

\[ \zeta(s) = \sum_{n=1}^{\infty} \frac{1}{n^s}, \quad \text{Re}(s) > 1 \]

欧拉乘积公式

\[ \zeta(s) = \prod_{p \text{ 素数}} \frac{1}{1 - p^{-s}}, \quad \text{Re}(s) > 1 \]

特殊值

\[ \zeta(2) = \frac{\pi^2}{6} \]
\[ \zeta(4) = \frac{\pi^4}{90} \]

黎曼ζ函数是解析数论中最重要的函数之一，它与素数分布有着密切的关系。

素数定理

渐近公式

\[ \pi(x) \sim \frac{x}{\ln x} \]

更精确的近似

\[ \pi(x) \sim \text{Li}(x) = \int_{2}^{x} \frac{dt}{\ln t} \]

误差估计

\[ \pi(x) = \text{Li}(x) + O(x e^{-c\sqrt{\ln x}}) \]

素数定理描述了素数分布的渐近行为，它指出小于等于x的素数个数π(x)近似等于x/ln x。

狄利克雷L函数

定义

\[ L(s, \chi) = \sum_{n=1}^{\infty} \frac{\chi(n)}{n^s}, \quad \text{Re}(s) > 1 \]

欧拉乘积公式

\[ L(s, \chi) = \prod_{p \text{ 素数}} \frac{1}{1 - \chi(p)p^{-s}}, \quad \text{Re}(s) > 1 \]

狄利克雷特征

\[ \chi: \mathbb{Z} \to \mathbb{C}, \quad \chi(mn) = \chi(m)\chi(n), \quad \chi(m+kq) = \chi(m) \]

狄利克雷L函数是黎曼ζ函数的推广，它与算术数列中的素数分布有关。
