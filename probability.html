概率统计公式

概率理论

基本概率公式

\[ P(A \cup B) = P(A) + P(B) - P(A \cap B) \]
\[ P(A \cap B) = P(A) \cdot P(B|A) = P(B) \cdot P(A|B) \]
\[ P(A|B) = \frac{P(A \cap B)}{P(B)} \]
\[ P(\overline{A}) = 1 - P(A) \]
\[ P\left(\bigcup_{i=1}^{n} A_i\right) \leq \sum_{i=1}^{n} P(A_i) \]
\[ P\left(\bigcap_{i=1}^{n} A_i\right) \geq 1 - \sum_{i=1}^{n} P(\overline{A_i}) \]

基本概率公式，包括并集概率、交集概率、条件概率、补集概率以及布尔不等式。

贝叶斯定理

\[ P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)} \]
\[ P(A_i|B) = \frac{P(B|A_i) \cdot P(A_i)}{\sum_{j=1}^{n} P(B|A_j) \cdot P(A_j)} \]

贝叶斯定理描述了在已知某事件B发生的条件下，事件A发生的概率。第二个公式是贝叶斯定理在多个互斥且穷尽的事件A₁, A₂, ..., Aₙ情况下的推广。

排列组合

\[ P(n,r) = \frac{n!}{(n-r)!} \]
\[ C(n,r) = \binom{n}{r} = \frac{n!}{r!(n-r)!} \]
\[ \binom{n}{r} = \binom{n}{n-r} \]
\[ \binom{n}{r} + \binom{n}{r-1} = \binom{n+1}{r} \]
\[ \sum_{r=0}^{n} \binom{n}{r} = 2^n \]
\[ \sum_{r=0}^{n} \binom{n}{r} x^r = (1+x)^n \]

排列组合公式，包括排列数P(n,r)和组合数C(n,r)的定义及性质。

概率公理

\[ P(A) \geq 0 \quad (\text{非负性}) \]
\[ P(\Omega) = 1 \quad (\text{规范性}) \]
\[ P\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i) \quad (\text{可列可加性}) \]

概率的三条公理：非负性、规范性和可列可加性。其中Ω是样本空间，A₁, A₂, ...是两两互斥的事件。

概率分布

离散概率分布

二项分布 B(n,p)

\[ P(X=k) = \binom{n}{k} p^k (1-p)^{n-k} \]
\[ E[X] = np, \quad Var(X) = np(1-p) \]

泊松分布 P(λ)

\[ P(X=k) = \frac{e^{-\lambda} \lambda^k}{k!} \]
\[ E[X] = \lambda, \quad Var(X) = \lambda \]

几何分布 Geom(p)

\[ P(X=k) = (1-p)^{k-1} p \]
\[ E[X] = \frac{1}{p}, \quad Var(X) = \frac{1-p}{p^2} \]

超几何分布 Hyper(N,K,n)

\[ P(X=k) = \frac{\binom{K}{k} \binom{N-K}{n-k}}{\binom{N}{n}} \]
\[ E[X] = \frac{Kn}{N}, \quad Var(X) = \frac{Kn(N-K)(N-n)}{N^2(N-1)} \]

常见离散概率分布的概率质量函数、期望和方差。

连续概率分布

正态分布 N(μ,σ²)

\[ f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2} \]
\[ E[X] = \mu, \quad Var(X) = \sigma^2 \]

均匀分布 U(a,b)

\[ f(x) = \begin{cases} \frac{1}{b-a}, & a \leq x \leq b \\ 0, & \text{其他} \end{cases} \]
\[ E[X] = \frac{a+b}{2}, \quad Var(X) = \frac{(b-a)^2}{12} \]

指数分布 Exp(λ)

\[ f(x) = \begin{cases} \lambda e^{-\lambda x}, & x \geq 0 \\ 0, & x < 0 \end{cases} \]
\[ E[X] = \frac{1}{\lambda}, \quad Var(X) = \frac{1}{\lambda^2} \]

伽马分布 Gamma(α,β)

\[ f(x) = \begin{cases} \frac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x}, & x > 0 \\ 0, & x \leq 0 \end{cases} \]
\[ E[X] = \frac{\alpha}{\beta}, \quad Var(X) = \frac{\alpha}{\beta^2} \]

卡方分布 χ²(k)

\[ f(x) = \begin{cases} \frac{1}{2^{k/2}\Gamma(k/2)} x^{k/2-1} e^{-x/2}, & x > 0 \\ 0, & x \leq 0 \end{cases} \]
\[ E[X] = k, \quad Var(X) = 2k \]

常见连续概率分布的概率密度函数、期望和方差。

多维概率分布

二维正态分布 N(μ₁,μ₂,σ₁²,σ₂²,ρ)

\[ f(x,y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left\{-\frac{1}{2(1-\rho^2)}\left[\left(\frac{x-\mu_1}{\sigma_1}\right)^2 - 2\rho\left(\frac{x-\mu_1}{\sigma_1}\right)\left(\frac{y-\mu_2}{\sigma_2}\right) + \left(\frac{y-\mu_2}{\sigma_2}\right)^2\right]\right\} \]

多项分布 Multinomial(n,p₁,p₂,...,pₖ)

\[ P(X_1=k_1,X_2=k_2,\ldots,X_k=k_k) = \frac{n!}{k_1!k_2!\cdots k_k!} p_1^{k_1}p_2^{k_2}\cdots p_k^{k_k}, \quad \sum_{i=1}^{k} k_i = n \]

边际分布与条件分布

\[ f_X(x) = \int_{-\infty}^{\infty} f_{X,Y}(x,y) dy \]
\[ f_{X|Y}(x|y) = \frac{f_{X,Y}(x,y)}{f_Y(y)} \]

多维概率分布，包括二维正态分布、多项分布以及边际分布和条件分布的定义。

统计推断

点估计

\[ \bar{X} = \frac{1}{n} \sum_{i=1}^{n} X_i \quad (\text{样本均值}) \]
\[ S^2 = \frac{1}{n-1} \sum_{i=1}^{n} (X_i - \bar{X})^2 \quad (\text{样本方差}) \]
\[ \hat{p} = \frac{X}{n} \quad (\text{比例估计}) \]
\[ \hat{\lambda} = \bar{X} \quad (\text{泊松参数估计}) \]

常见参数的点估计方法，包括样本均值、样本方差、比例估计和泊松参数估计。

区间估计

均值的置信区间（σ已知）

\[ \bar{X} \pm z_{\alpha/2} \frac{\sigma}{\sqrt{n}} \]

均值的置信区间（σ未知）

\[ \bar{X} \pm t_{\alpha/2,n-1} \frac{S}{\sqrt{n}} \]

比例的置信区间

\[ \hat{p} \pm z_{\alpha/2} \sqrt{\frac{\hat{p}(1-\hat{p})}{n}} \]

方差的置信区间

\[ \left( \frac{(n-1)S^2}{\chi^2_{\alpha/2,n-1}}, \frac{(n-1)S^2}{\chi^2_{1-\alpha/2,n-1}} \right) \]

常见参数的区间估计方法，包括均值、比例和方差的置信区间。

假设检验

均值检验（σ已知）

\[ Z = \frac{\bar{X} - \mu_0}{\sigma/\sqrt{n}} \]

均值检验（σ未知）

\[ t = \frac{\bar{X} - \mu_0}{S/\sqrt{n}} \]

比例检验

\[ Z = \frac{\hat{p} - p_0}{\sqrt{p_0(1-p_0)/n}} \]

方差检验

\[ \chi^2 = \frac{(n-1)S^2}{\sigma_0^2} \]

两个均值之差的检验（独立样本）

\[ t = \frac{(\bar{X}_1 - \bar{X}_2) - (\mu_1 - \mu_2)}{S_p \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}} \]
\[ S_p = \sqrt{\frac{(n_1-1)S_1^2 + (n_2-1)S_2^2}{n_1 + n_2 - 2}} \]

常见参数的假设检验方法，包括均值、比例、方差的检验统计量以及两个均值之差的检验。
