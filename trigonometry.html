三角函数公式

基本三角函数

三角函数定义

\[ \sin\theta = \frac{对边}{斜边} = \frac{y}{r} \]
\[ \cos\theta = \frac{邻边}{斜边} = \frac{x}{r} \]
\[ \tan\theta = \frac{对边}{邻边} = \frac{y}{x} = \frac{\sin\theta}{\cos\theta} \]
\[ \cot\theta = \frac{邻边}{对边} = \frac{x}{y} = \frac{\cos\theta}{\sin\theta} = \frac{1}{\tan\theta} \]
\[ \sec\theta = \frac{斜边}{邻边} = \frac{r}{x} = \frac{1}{\cos\theta} \]
\[ \csc\theta = \frac{斜边}{对边} = \frac{r}{y} = \frac{1}{\sin\theta} \]

三角函数的基本定义，基于直角三角形的边长比例或单位圆上点的坐标。其中 \(x, y\) 是点的坐标，\(r\) 是点到原点的距离。

三角函数的周期性

\[ \sin(\theta + 2\pi) = \sin\theta \]
\[ \cos(\theta + 2\pi) = \cos\theta \]
\[ \tan(\theta + \pi) = \tan\theta \]
\[ \cot(\theta + \pi) = \cot\theta \]
\[ \sec(\theta + 2\pi) = \sec\theta \]
\[ \csc(\theta + 2\pi) = \csc\theta \]

三角函数的周期性性质。正弦、余弦、正割和余割函数的周期是 \(2\pi\)，而正切和余切函数的周期是 \(\pi\)。

三角函数的奇偶性

\[ \sin(-\theta) = -\sin\theta \quad (\text{奇函数}) \]
\[ \cos(-\theta) = \cos\theta \quad (\text{偶函数}) \]
\[ \tan(-\theta) = -\tan\theta \quad (\text{奇函数}) \]
\[ \cot(-\theta) = -\cot\theta \quad (\text{奇函数}) \]
\[ \sec(-\theta) = \sec\theta \quad (\text{偶函数}) \]
\[ \csc(-\theta) = -\csc\theta \quad (\text{奇函数}) \]

三角函数的奇偶性。余弦和正割是偶函数，其他三角函数都是奇函数。

三角恒等式

基本恒等式

\[ \sin^2\theta + \cos^2\theta = 1 \]
\[ 1 + \tan^2\theta = \sec^2\theta \]
\[ 1 + \cot^2\theta = \csc^2\theta \]
\[ \tan\theta = \frac{\sin\theta}{\cos\theta} \]
\[ \cot\theta = \frac{\cos\theta}{\sin\theta} \]
\[ \sec\theta = \frac{1}{\cos\theta} \]
\[ \csc\theta = \frac{1}{\sin\theta} \]

三角函数的基本恒等式，包括勾股恒等式和倒数关系。

和角公式

\[ \sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta \]
\[ \sin(\alpha - \beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta \]
\[ \cos(\alpha + \beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta \]
\[ \cos(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta \]
\[ \tan(\alpha + \beta) = \frac{\tan\alpha + \tan\beta}{1 - \tan\alpha\tan\beta} \]
\[ \tan(\alpha - \beta) = \frac{\tan\alpha - \tan\beta}{1 + \tan\alpha\tan\beta} \]

和角公式用于计算两个角的和或差的三角函数值。

倍角公式

\[ \sin(2\theta) = 2\sin\theta\cos\theta \]
\[ \cos(2\theta) = \cos^2\theta - \sin^2\theta = 2\cos^2\theta - 1 = 1 - 2\sin^2\theta \]
\[ \tan(2\theta) = \frac{2\tan\theta}{1 - \tan^2\theta} \]
\[ \sin(3\theta) = 3\sin\theta - 4\sin^3\theta \]
\[ \cos(3\theta) = 4\cos^3\theta - 3\cos\theta \]
\[ \tan(3\theta) = \frac{3\tan\theta - \tan^3\theta}{1 - 3\tan^2\theta} \]

倍角公式用于计算角的倍数的三角函数值。

半角公式

\[ \sin\left(\frac{\theta}{2}\right) = \pm\sqrt{\frac{1 - \cos\theta}{2}} \]
\[ \cos\left(\frac{\theta}{2}\right) = \pm\sqrt{\frac{1 + \cos\theta}{2}} \]
\[ \tan\left(\frac{\theta}{2}\right) = \pm\sqrt{\frac{1 - \cos\theta}{1 + \cos\theta}} = \frac{1 - \cos\theta}{\sin\theta} = \frac{\sin\theta}{1 + \cos\theta} \]

半角公式用于计算角的一半的三角函数值。符号取决于 \(\frac{\theta}{2}\) 所在的象限。

和差化积公式

\[ \sin\alpha + \sin\beta = 2\sin\left(\frac{\alpha + \beta}{2}\right)\cos\left(\frac{\alpha - \beta}{2}\right) \]
\[ \sin\alpha - \sin\beta = 2\cos\left(\frac{\alpha + \beta}{2}\right)\sin\left(\frac{\alpha - \beta}{2}\right) \]
\[ \cos\alpha + \cos\beta = 2\cos\left(\frac{\alpha + \beta}{2}\right)\cos\left(\frac{\alpha - \beta}{2}\right) \]
\[ \cos\alpha - \cos\beta = -2\sin\left(\frac{\alpha + \beta}{2}\right)\sin\left(\frac{\alpha - \beta}{2}\right) \]

和差化积公式用于将两个三角函数的和或差转换为乘积形式。

积化和差公式

\[ \sin\alpha\cos\beta = \frac{1}{2}[\sin(\alpha + \beta) + \sin(\alpha - \beta)] \]
\[ \cos\alpha\sin\beta = \frac{1}{2}[\sin(\alpha + \beta) - \sin(\alpha - \beta)] \]
\[ \cos\alpha\cos\beta = \frac{1}{2}[\cos(\alpha + \beta) + \cos(\alpha - \beta)] \]
\[ \sin\alpha\sin\beta = \frac{1}{2}[\cos(\alpha - \beta) - \cos(\alpha + \beta)] \]

积化和差公式用于将两个三角函数的乘积转换为和或差的形式。

三角形定理

正弦定理

\[ \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R \]

正弦定理描述了三角形的边长与对角的正弦值之间的关系。其中 \(a, b, c\) 是三角形的边长，\(A, B, C\) 分别是对应边的对角，\(R\) 是三角形的外接圆半径。

余弦定理

\[ a^2 = b^2 + c^2 - 2bc\cos A \]
\[ b^2 = a^2 + c^2 - 2ac\cos B \]
\[ c^2 = a^2 + b^2 - 2ab\cos C \]

余弦定理描述了三角形的边长与夹角的余弦值之间的关系。其中 \(a, b, c\) 是三角形的边长，\(A, B, C\) 分别是对应边的对角。

三角形面积公式

\[ A = \frac{1}{2}bh \]
\[ A = \frac{1}{2}ab\sin C = \frac{1}{2}bc\sin A = \frac{1}{2}ac\sin B \]
\[ A = \sqrt{s(s-a)(s-b)(s-c)} \quad (\text{海伦公式}) \]
\[ A = \frac{abc}{4R} \]
\[ A = rs \]

三角形面积的多种计算公式。其中 \(b\) 是底边长，\(h\) 是高，\(a, b, c\) 是三角形的边长，\(A, B, C\) 分别是对应边的对角，\(s = \frac{a+b+c}{2}\) 是半周长，\(R\) 是外接圆半径，\(r\) 是内切圆半径。
